How It Works

This application uses a pre-trained convolutional neural network to detect emotions such as:

  • Angry
  • Happy
  • Sad
  • Neutral
  • Fearful
  • Disgusted
  • Surprised

Face detection is powered by OpenCV Haar cascades, and emotion classification is achieved with TensorFlow.
